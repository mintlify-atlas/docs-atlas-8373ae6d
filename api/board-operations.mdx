---
title: Board Operations
description: API reference for creating, updating, removing, and retrieving boards in TaskForge Studio
---

The Board Operations API provides mutations and queries for managing boards in your organization.

## create

Creates a new board in the specified organization.

<ParamField path="orgId" type="string" required>
  The organization ID where the board will be created
</ParamField>

<ParamField path="title" type="string" required>
  The title of the board
</ParamField>

### Returns

Returns the board ID of the newly created board.

<ResponseField name="boardId" type="Id<'boards'>">
  The unique identifier for the created board
</ResponseField>

### Response Schema

<Expandable title="Board Object">
  <ResponseField name="_id" type="Id<'boards'>">
    Unique board identifier
  </ResponseField>
  
  <ResponseField name="title" type="string">
    Board title
  </ResponseField>
  
  <ResponseField name="orgId" type="string">
    Organization ID
  </ResponseField>
  
  <ResponseField name="authorId" type="string">
    User ID of the board creator
  </ResponseField>
  
  <ResponseField name="authorName" type="string">
    Name of the board creator
  </ResponseField>
  
  <ResponseField name="imageUrl" type="string">
    Randomly assigned placeholder image URL
  </ResponseField>
  
  <ResponseField name="_creationTime" type="number">
    Timestamp when the board was created
  </ResponseField>
</Expandable>

### Example

```typescript
const boardId = await convex.mutation(api.board.create, {
  orgId: "org_2abc123def456",
  title: "Q4 Product Roadmap"
});
```

### Errors

- **Unauthorized**: Thrown when the user is not authenticated

---

## update

Updates the title of an existing board.

<ParamField path="id" type="Id<'boards'>" required>
  The ID of the board to update
</ParamField>

<ParamField path="title" type="string" required>
  The new title for the board (max 60 characters)
</ParamField>

### Returns

Returns `null` on successful update.

### Example

```typescript
await convex.mutation(api.board.update, {
  id: boardId,
  title: "Updated Q4 Roadmap"
});
```

### Errors

- **Unauthorized**: Thrown when the user is not authenticated
- **Title is required**: Thrown when the title is empty or only whitespace
- **Title cannot be more than 60 characters**: Thrown when the title exceeds the maximum length

---

## remove

Deletes a board and removes it from all user favorites.

<ParamField path="id" type="Id<'boards'>" required>
  The ID of the board to delete
</ParamField>

### Returns

Returns `void` on successful deletion.

### Example

```typescript
await convex.mutation(api.board.remove, {
  id: boardId
});
```

### Behavior

- Automatically removes the board from any user's favorites before deletion
- Permanently deletes the board from the database

### Errors

- **Unauthorized**: Thrown when the user is not authenticated

---

## get

Retrieves a single board by its ID.

<ParamField path="id" type="Id<'boards'>" required>
  The ID of the board to retrieve
</ParamField>

### Returns

<ResponseField name="board" type="Board | null">
  The board object if found, or `null` if the board doesn't exist
</ResponseField>

### Response Schema

<Expandable title="Board Object">
  <ResponseField name="_id" type="Id<'boards'>">
    Unique board identifier
  </ResponseField>
  
  <ResponseField name="title" type="string">
    Board title
  </ResponseField>
  
  <ResponseField name="orgId" type="string">
    Organization ID
  </ResponseField>
  
  <ResponseField name="authorId" type="string">
    User ID of the board creator
  </ResponseField>
  
  <ResponseField name="authorName" type="string">
    Name of the board creator
  </ResponseField>
  
  <ResponseField name="imageUrl" type="string">
    Board's placeholder image URL
  </ResponseField>
  
  <ResponseField name="_creationTime" type="number">
    Timestamp when the board was created
  </ResponseField>
</Expandable>

### Example

```typescript
const board = await convex.query(api.board.get, {
  id: boardId
});

if (board) {
  console.log(board.title);
}
```
